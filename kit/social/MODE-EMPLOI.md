## Fichiers inclus

|---|---|---|
| `post-facebook-paysage.html` | 1200×630 px | Post Facebook paysage, partage LinkedIn |

| `post-story-1080x1920.html` | 1080×1920 px | Story Facebook / Instagram |

## Comment créer les images (Chrome)

1. Ouvrir le fichier HTML dans **Chrome**
2. Ouvrir les DevTools : `F12` ou `Cmd+Opt+I`
3. Cliquer sur l’icône téléphone (« Toggle device toolbar ») ou `Ctrl+Shift+M`
4. Entrer les dimensions exactes en haut (ex: 1200 × 630)
5. `Ctrl+Shift+P` → taper **« screenshot »** → **Capture full size screenshot**
6. L’image PNG est téléchargée automatiquement

## Où publier chaque format

| Format | Plateforme | Note |
|---|---|---|

| Carré 1080×1080 | Facebook, Instagram | Plus de visibilité dans le feed |
| Story 1080×1920 | Facebook Story, Instagram Story | 24h, très visible |
